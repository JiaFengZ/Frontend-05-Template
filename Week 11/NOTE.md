## specificity
* 通配符(0, 0, 0, 0)
* 标签(0, 0, 0, 1)
* class (0, 0, 1, 0)
* 属性 (0, 0, 1, 0)
* 伪类 (0, 0, 1, 0)
* id (0, 1, 0, 0)
* important (1, 0, 0, 0)

### 例子
* div#a.b .c[id=x] ->  (0, 1, 3, 1)
* #a:not(#b) -> (0, 2, 0, 0)
* *.a -> (0, 0, 1, 0)
* div.a -> (0, 0, 1, 1)

## 为什么 first-letter 可以设置 float 之类的，而 first-line 不行
first-line需要根据布局结果计算得得出选择结果，如果设置float，影响布局结果，就没办法确定first-line
